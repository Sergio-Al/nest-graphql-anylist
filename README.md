# Instructions

# Dev

1. Copy `.env.template` to `.env` and fill in the values
2. Install the dependencies
```
npm install
```
3. Run the image with docker-compose 
```
docker-compose up -d
```
4. Run the project
`npm run start:dev`
5. Open `http://localhost:3000/graphql` in your browser
